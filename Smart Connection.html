Smart Connection

In de connection workshops hebben we, de minor klas, ons verdiept in de wereld van HTML, CSS, JavaScript en API. In het verleden heb ik vaker websites aangepast op HTML en CSS, maar dit was in programma's als WordPress. Van begin af aan een website 'schrijven' was dus wel nieuw voor mij.
Ik heb de filmpjes bekeken op Moodle over HTML, CSS en JAVA en de WebDev opdrachten gemaakt. Daarbij heb ik vooral veel geprobeerd op deze website, ondertussen is deze website al groen, roze en multicolor geweest.

Hieronder laat ik verschillende voorbeelden zien van wat ik heb geleerd, in mijn favoriete thema: eten!

HTML & CSS

Als basis voor de website heb ik een template gebruikt van Bootstrap. Met een CSS pagina daar boven op heb ik de opmaak kunnen veranderen. Zo heb ik de kopjes en links verschillende kleuren, schaduwen en diktes gegeven.
Daarnaast denk ik dat ik met het maken van deze website een aardige HTML basiskennis heb gevormd voor mezelf.
